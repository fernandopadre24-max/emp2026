# **App Name**: Gestão Fácil

## Core Features:

- Registro de Empréstimos: Permitir o registro de novos empréstimos com detalhes como valor, taxa de juros, data de vencimento e mutuário.
- Acompanhamento de Pagamentos: Registrar e acompanhar os pagamentos efetuados, atualizando o status do empréstimo.
- Cálculo de Juros: Calcular automaticamente os juros a serem pagos em cada parcela do empréstimo.
- Geração de Alertas: Gerar alertas automáticos de vencimento de parcelas para mutuários e gestores.
- Relatórios Financeiros: Gerar relatórios financeiros com o resumo dos empréstimos ativos, pagamentos pendentes e recebimentos efetuados.
- Análise de Crédito (AI): Utilizar um modelo de IA como tool para avaliar o risco de crédito de novos mutuários, utilizando dados históricos e informações disponíveis.

## Style Guidelines:

- Cor primária: Azul claro (#A7D1AB) para transmitir confiança e segurança.
- Cor de fundo: Branco acinzentado suave (#F2F4F7) para um visual limpo e moderno.
- Cor de destaque: Verde (#61B33B) para botões de ação e notificações positivas.
- Fonte: 'Inter', uma sans-serif para transmitir uma aparência moderna, tecnológica e objetiva.
- Ícones claros e simples para representar as diferentes funcionalidades e informações do aplicativo.
- Design limpo e intuitivo, com informações organizadas de forma clara e fácil de entender.
- Transições suaves e sutis entre as telas para melhorar a experiência do usuário.
